# 2026-03-02 - 创模云科项目

## 项目背景
用户分享了"创模云科"公司的网站前端代码，这是一个 3D 模型交易与 AI 生成平台。

## 平台核心功能

### 1. AI 生成区 (GenerateSection)
- 图片转 3D 模型功能
- 上传 2D 设计图，AI 引擎生成高精度 3D 模型预览
- 支持 JPG, PNG, WEBP (最大 20MB)
- 展示实时生成案例（原图 vs 3D 预览对比）

### 2. 模型交易区 (TradingSection)
- 3D 模型市场，创作者可出售模型
- 搜索与筛选功能（最新、热门、免费、机甲、角色等标签）
- 模型卡片展示：价格、作者、标签、点赞、评论
- 创作者榜单（本周成交额排名）

### 3. 需求悬赏大厅 (RequirementsSection)
- 用户发布定制需求，创作者接单
- 悬赏金额展示（按金额排序）
- 需求状态：open / urgent / closed
- 悬赏金主榜（发布者的总悬赏金额排名）

### 4. 社区讨论区 (CommunitySection)
- 作品展示与分享
- 对比图展示（3D 模型 vs 实物成品）
- 热门话题标签
- 推荐关注创作者
- 排序：最新 / 热门

### 5. 供应商列表 (SupplierSection)
- 连接线下 3D 打印工厂
- 供应商信息：月成交量、评分、平均报价、专长、工期
- 快速估价工具（上传模型文件，选择材质/填充率）
- 排序：月成交量 / 好评优先 / 价格

## 技术栈
- React (函数组件 + Hooks)
- Tailwind CSS
- lucide-react 图标库
- 单文件组件结构（待模块化）
